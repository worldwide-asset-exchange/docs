# Tokens

This section is for teams and creators planning fungible token launches on WAX.

## Key Decisions

- what the token is for
- who can issue it
- total supply or issuance rules
- decimal precision
- treasury and distribution plans
- whether minting stays open after launch

## Launch Considerations

Token launches are not just a contract task. You should also plan for:

- issuer account security
- treasury management
- user communication and token utility
- exchange or market availability
- wallet and ecosystem support
