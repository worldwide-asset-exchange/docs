# Tools and Services

This section helps creators find launch support, community resources, and service providers that can help bring a WAX project to market.

## What This Covers

Creator support on WAX can include:

- community promotion
- launch support
- ecosystem tooling
- funding opportunities
- managed infrastructure and API services

## Start Here

- [Community Resources](/create/community)
- [WAX Labs](/create/wax-labs/)
- [WAX API Services](/build/wax-api-services)

## Choosing the Right Path

Use community resources when you need visibility and ecosystem support. Use WAX Labs when your project needs funding and milestone-based review. Use API services when you need reliability without self-hosting infrastructure.
