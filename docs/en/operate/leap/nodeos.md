# nodeos

`nodeos` is the core daemon that runs a WAX node.

## Typical Roles

- local development node
- chain API node
- peer node
- state-history node
- specialized infrastructure service node

## Why It Matters

Most operational design choices eventually come back to how `nodeos` is configured, isolated, monitored, and connected to the rest of your stack.
