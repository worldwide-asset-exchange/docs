# keosd

`keosd` is the wallet component used with command-line workflows for key and signing management.

## Common Uses

- storing keys for command-line workflows
- unlocking wallet state for signed actions
- supporting local deployment and operational tasks

## Operational Reminder

Treat `keosd` and anything that can unlock it as sensitive operational material. Avoid casual reuse of high-authority keys.
