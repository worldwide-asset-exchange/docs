# Getting Started with WAX

This section is organized around the first tasks most WAX users run into: choosing a wallet, funding an account, understanding resources, and learning how governance works.

## Suggested Order

1. [Choose a wallet](/learn/getting-started/choosing-a-wallet)
2. [Set up My Cloud Wallet](/learn/getting-started/mycloudwallet/) or [Anchor](/learn/getting-started/anchor/)
3. [Review security and recovery basics](/learn/getting-started/security-and-recovery)
4. [Get WAXP and fund an account](/learn/getting-started/get-waxp-and-fund-account)
5. [Understand PowerUp, CPU, NET, and RAM](/learn/getting-started/powerup-and-resources)
6. [Learn staking and governance basics](/learn/getting-started/staking-and-governance)
7. [Explore marketplaces](/learn/getting-started/marketplaces)
