# Choosing a Wallet

No single wallet is best for every WAX user. Choose based on the kind of access, control, and recovery model you need.

## Recommended by Use Case

### New to WAX

Start with [My Cloud Wallet](/learn/getting-started/mycloudwallet/).

It offers the easiest onboarding path and is the simplest way to start using games, marketplaces, and other WAX apps without learning the full key-management model on day one.

### Want More Direct Control

Use [Anchor](/learn/getting-started/anchor/).

Anchor is better when you want local wallet management, deeper account controls, and a workflow that fits power users or developers.

### Mostly Using Games and Apps

Consider My Cloud Wallet or Wombat first.

Both are designed around regular app interaction. The better choice depends on your preferred device, sign-in flow, and the apps you use most often.

### Building or Operating on WAX

Use Anchor and/or `cleos`.

Developers and operators typically need better control over permissions, signing, account management, and automation than a browser-first wallet provides.

## Decision Checklist

Pick the wallet that best matches the statements below:

- I want the fastest consumer onboarding: My Cloud Wallet
- I want stronger desktop control and advanced account features: Anchor
- I want a game-friendly multi-chain wallet: Wombat
- I want command-line control and scripting: cleos

## After You Choose

- Review [Security and Recovery](/learn/getting-started/security-and-recovery)
- If you chose My Cloud Wallet, continue to [My Cloud Wallet](/learn/getting-started/mycloudwallet/)
- If you chose Anchor, continue to [Anchor](/learn/getting-started/anchor/)
