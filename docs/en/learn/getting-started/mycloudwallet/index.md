# My Cloud Wallet

My Cloud Wallet is WAX's user-friendly wallet for signing in, managing digital assets, and using WAX-based apps. The latest version uses passkeys for sign-in and supports recovery with a 12-word mnemonic phrase.

::: info NOTE
Cloud Wallet Mobile is currently no longer available.
:::

## What is My Cloud Wallet?

My Cloud Wallet is designed to make blockchain interactions simpler for everyday users. It handles account access through modern passkeys, supports asset management for WAXP and NFTs, and connects directly to games, marketplaces, and other dApps in the WAX ecosystem.

### Key Features

- **Passkey Sign-In**: Use Face ID, Touch ID, fingerprint, or device PIN instead of a traditional password
- **Mnemonic Recovery**: Recover access or add a new device using your 12-word recovery phrase
- **Secure Asset Storage**: Safely store WAXP tokens, NFTs, and other digital collectibles
- **dApp Integration**: Seamlessly interact with WAX-based games and applications
- **Vault Sessions**: Keep a signing session open for smoother repeated transactions

## Getting Started

### Creating Your Account

1. Visit [My Cloud Wallet](https://www.mycloudwallet.com)
2. Choose to create a new account
3. Save your 12-word mnemonic phrase securely and offline
4. Verify the requested recovery words
5. Choose your 12-character WAX account name
6. Create a passkey on your device or by using your phone

## Why Choose My Cloud Wallet?

### For Users

- **No Technical Knowledge Required**: Perfect for newcomers to blockchain
- **Simple Passkey Security**: Sign in without memorizing a password
- **Cross-Platform Access**: Works seamlessly on desktop and mobile web
- **Recovery Options**: Restore access with your mnemonic phrase if you lose a device

### For Gamers

- **Instant Access**: Jump into games without complex wallet setup
- **NFT Management**: Easily view and trade your gaming collectibles
- **Reward Tracking**: Monitor your staking rewards and token balances
- **Faster Signing**: Use the Vault for smoother repeated actions in supported dApps

## Security & Trust

My Cloud Wallet prioritizes security while maintaining ease of use:

- **Passkey Protection**: Your private key is protected by your device's passkey system
- **Mnemonic Backups**: Your 12-word phrase is your master recovery method
- **Optional Advanced Control**: Legacy users can still review claiming options if they need direct key ownership
