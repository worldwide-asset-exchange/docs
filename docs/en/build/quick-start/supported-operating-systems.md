# Supported Operating Systems

WAX development and operations tooling is most commonly used from Linux-based environments.

## Recommended Environments

### Linux

Linux remains the most common environment for local contract tooling, node software, and server operations.

### Windows with WSL

Windows users typically get the best experience by running Linux tooling through WSL when they need native Antelope-style development or operational workflows.

### macOS

macOS can work well for application development and some local workflows, but you should validate toolchain compatibility early if you need lower-level contract or node tooling.

## Practical Guidance

- app developers can often work productively without a full local chain
- contract developers should validate compiler and toolchain requirements early
- operators should prefer Linux for production and automation work
