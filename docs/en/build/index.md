# Build on WAX

The Build section is organized around the major jobs developers need to do on WAX:

- learn the core concepts that shape account, permission, and transaction design
- get a local toolchain working
- write and deploy smart contracts
- integrate wallets and app sessions
- choose APIs and endpoints for production apps
