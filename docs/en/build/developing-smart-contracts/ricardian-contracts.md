# Ricardian Contracts

Ricardian contracts provide human-readable terms that sit alongside contract actions. They help explain what an action is meant to do and what parties are agreeing to when they use it.

## Why They Matter

- they improve clarity for users and auditors
- they document action intent alongside contract code
- they help separate technical execution from human-readable meaning

## WAX Context

On WAX, Ricardian material is part of a contract-development workflow rather than a standalone documentation concern. Builders should treat it as part of contract quality, not as an afterthought.
