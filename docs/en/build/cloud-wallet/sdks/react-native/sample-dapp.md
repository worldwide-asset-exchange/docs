# Sample dApp
1. Clone repository https://github.com/worldwide-asset-exchange/sdk-react-native
2. Go to `dapp-example` folder
3. Execute command:
```sh
yarn run ios
```
Notes: To switch between stg and production mode, Long press to Network header(Wax Mainnet)
