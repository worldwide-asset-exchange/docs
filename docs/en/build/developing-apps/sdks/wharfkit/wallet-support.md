# Wallet Support

WharfKit is useful because it lets builders support more than one wallet path without building a separate integration stack for each one.

## WAX Wallets to Plan For

- My Cloud Wallet
- Anchor
- Wombat

## How to Choose Your Supported Set

Support the wallets your audience already uses.

- consumer apps often need My Cloud Wallet first
- advanced tools often need Anchor support
- game and mobile-heavy audiences may expect Wombat support

## Product Guidance

- do not overload users with every wallet unless there is a real need
- keep wallet copy and support messaging explicit
- test login, restore, and signing behavior per wallet instead of assuming parity
