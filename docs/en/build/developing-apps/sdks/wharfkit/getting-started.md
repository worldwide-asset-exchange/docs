# WharfKit Getting Started

Use WharfKit when you need a modern WAX app integration path for login, sessions, and transaction signing.

## Typical Setup Flow

1. choose the wallets you want to support
2. configure the WAX chain and endpoint settings
3. initialize session management in your app
4. present a clear login path to the user
5. test signing, restore, and logout behavior

## Good First Use Cases

- browser apps
- dashboards and account tools
- game frontends
- backend-assisted apps that still rely on user signatures
