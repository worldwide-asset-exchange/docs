# Tokens

Tokens on WAX include the native token WAXP and contract-issued fungible tokens used by apps, games, and services.

## WAXP

WAX is the native token used across the network for account activity, ecosystem participation, and governance-related workflows.

## Contract Tokens

Builders can also deploy fungible-token contracts for:

- in-app currencies
- reward systems
- utility tokens
- creator and community economies

## Why Tokens Matter

Token design affects:

- resource planning
- treasury and reward models
- marketplace behavior
- governance and staking patterns
